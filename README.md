# A Little Bit About This Project
This portfolio website was designed and created from scratch using HTML, CSS and some JavaScript for my university project in web development. Anything I have used is referenced in the footer of the website. 

# HTML
The html is a very simple design where the user can press a button and it scrolls to the next section of the website. I chose this as I thought it would make it easier for the reader.

# CSS
I have attempted to keep the CSS styling of this website more modern looking and utilise a colour palette that works well together. I wanted vibrant colours and effects to captivate the reader. The carousel is a nice touch in my opinion for viewing my projects and keeping the website organised.

# JavaScript
The JavaScript was a challenge to write as a beginner and most of it was taken from my university lectures. This is one area that I need to improve on for a more polished website.
